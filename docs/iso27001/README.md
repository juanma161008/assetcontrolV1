# ISO 27001 - ISMS Pack (AssetControl)

This folder contains the Information Security Management System (ISMS) starter pack for AssetControl.
It is aligned with ISO/IEC 27001:2022 requirements and Annex A controls.

Important:
- This documentation does not represent an official certification.
- It is a working set of policies, procedures, and records to achieve compliance readiness.
- Update owners, dates, and evidence as you implement controls.
